Concepts de base¶

Rappels :

Un "graphique" est strucutr de la manire suivante :

  • Figure est l'objet de plus haut niveau : "coquille externe" du graphique
  • Axes contient l'objet graphique en lui-mme
  • Axis est un axe du graphique (contrle des labels, xmin, xmax, etc.)
  • Artist est chaque lment dessin sur un graphique (ligne, titre, etc)
  • Canvas est la zone de trac du graphe

Les donnes tracer doivent tre de type array de numpy. (np.array).

Si les donnes sont sous formes de squences, elles sont converties en interne en array.

Le module pyplot cre automatiquement (et de faon transparente) les objets Figure et Axes ncessaires la construction d'un graphique. Chaque "lment" qu'on rajoute par la suite (ex : plt.plot() ou plt.title()) s'ajoute la Figure et l'Axes actif.

Pour visualiser la figure cre, il faut utiliser la commande plt.show()

Création d'un graphique¶

La commande de base est plt.plot(x, y, formattage)

x et y tant l'abscisse et l'ordonne sous forme de np.array.

Plusieurs Axes sur une Figure¶

Un Axes est plac dans un "grille" de $N_l$ lignes et $N_c$ colonnes, avec un identifiant $id$ (entier de 1 $N_l \times N_c$), sous le modle subplot(Nl, Nc, id)

In [4]:
plt.figure(5)          # cration de la Figure
plt.subplot(2, 3, 1)   # premier Axes avec une Figure contenant 2 lignes, 3 colonnes
plt.plot(x, y)
ax_id = plt.subplot(2, 3, 2)   # ax_id permet de garder une rfrence  cet Axes      
plt.plot(x**2, y)
plt.subplot(2, 3, 3)            
plt.subplot(2, 3, 4)
plt.plot([i for i in range(7, 12)], [1, -1, 2,6, 8])
plt.subplot(2, 3, 5, projection='polar')
plt.subplot(2, 3 ,6 , sharex=ax_id, facecolor='red') # partage un axe des abscisses commun avec le graphe ax_id

plt.show()

Manipulation des Axes (programmation orientée objet)¶

En gardant une rfrence vers les objets crs (Figure ou Axes), on peut modifier leurs attributs et "embellir" le graphique.

In [6]:
# backend permettant de "manipuler" le graphique
%matplotlib notebook 

# cration des donnes  tracer
t = np.arange(0.0, 2.0, 0.01)
s = 1 + np.sin(2 * np.pi * t)

fig, ax = plt.subplots() # on garde une rfrence aux objets crs
ax.plot(t, s) # mthode plot de l'objet Axes

ax.set(xlabel='temps (s)', ylabel='tension (mV)', title='Tension alternative')  # mthode set de l'objet Axes
ax.grid() # mthode grid de l'objet Axes

fig.savefig("test.png") # enregistre l'image avec la mthode save de Figure
plt.show()

Fonctions mathématiques de Numpy utiles¶

Numpy contient toutes les fonctions usuelles de math et peut appliquer ces fonctions l'ensemble des donnes d'un Array numpy.

  • Constantes : np.pi, np.e
  • Fonctions trigo : np.cos, np.sin, np.tan
  • Logarithme : np.log (log nprien), np.log10 (log dcimal)
