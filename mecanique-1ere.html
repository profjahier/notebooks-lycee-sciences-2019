Calcul du vecteur vitesse¶

$v_{x}[i] = \dfrac{x_{[i+1]} - x_{[i-1]}} {t_{[i+1]} - t_{[i-1]}}$

$v_{y}[i] = \dfrac{y_{[i+1]} - y_{[i-1]}} {t_{[i+1]} - t_{[i-1]}}$

On donne les valeurs suivantes "aux extrmits" :

$v_{x}[0] = \dfrac{x_{[1]} - x_{[0]}} {t_{[1]} - t_{[0]}}$

$v_{y}[-1] = \dfrac{y_{[-1]} - y_{[-2]}} {t_{[-1]} - t_{[-2]}}$

In [5]:
def vitesse(x, y, t):
    """ calcul des coordonnes de v """
    vx = [(x[1] - x[0]) / (t[1] - t[0])]
    for i in range(1, len(x)-1):
        vx.append((x[i+1] - x[i-1]) / (t[i+1] - t[i-1]))
    vx.append((x[-1] - x[-2]) / (t[-1] - t[-2]))

    vy = [(y[1] - y[0]) / (t[1] - t[0])]
    for i in range(1, len(y)-1):
        vy.append((y[i+1] - y[i-1]) / (t[i+1] - t[i-1]))
    vy.append((y[-1] - y[-2]) / (t[-1] - t[-2]))
    return vx, vy

Calcul du vecteur variation de vitesse¶

$Dv_x[i] = v_x[i+1] - v_x[i-1]$

$Dv_y[i] = v_y[i+1] - v_y[i-1]$

On donne les valeurs suivantes "aux extrmits" :

$Dv_x[0] = v_x[1] - v_x[0]$

$Dv_x[-1] = v_x[-1] - v_x[-2]$

In [8]:
def var_vitesse(vx, vy):
    """ calcul des coordonnes de Delta_v """
    Dvx = [vx[1] - vx[0]]
    for i in range(1, len(vx)-1):
        Dvx.append(vx[i+1] - vx[i-1])
    Dvx.append(vx[-1] - vx[-2])

    Dvy = [vy[1] - vy[0]]
    for i in range(1, len(y)-1):
        Dvy.append(vy[i+1] - vy[i-1])
    Dvy.append(vy[-1] - vy[-2])
    return Dvx, Dvy
